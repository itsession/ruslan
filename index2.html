Инструкция

Подготовительный этап

  1. определить среду разработки(выбрал Visual Studio Code)
  2. создать фаил index.html
  3. создать базовую структуру документа html (!+tap)
  4. создать элементы на строничке

Создание элементов на странице

  1. h1-h6 создание заголовков разного размера на странице (они располагаются по центру автоматически)
  2. Тэг img создает картинку на странице. Для этого внутрь тега img необходимо записать атрибут src="сылка на картинку"
  3. Тэг а создает сылку на какой нибудь сайт или на другую страницу моего сайта. Для этого внутрь тега а необходимо записать атрибут href="сылка на страницу"
  4. div создает блок на странице. Нужен для того чтобы размещать внутри него другие элементы на странице, а затем перемещать или позиционировать этот блок с элементами так как нам необходимо, в нужном месте страницы или в нужной форме с нужными нам стилями.
  5. Ol создает нумерованный список на странице. Для этого внутрь тэга Ol необходимо добавить теги li. Каждый тег li это отдельный элемент нумерованного списка.
